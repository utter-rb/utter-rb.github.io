About

In this document, I will demonstrate mushin's features such as code-generation of domain frameworks and domain middleware.

Installation

Install mushin in your ruby enviroment normally or via Bundler. 

$ gem install mushin

Create your Domain Framework

Now we will create a new Domain framework for the gamification domain. We will call it "GameOn".
